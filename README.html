📖 How to Use These Notes
Recommended study approach for interview prep

Study Order (Recommended)

1
Read notes first — Open 01_*_notes.md. Read sections 1–4 only (Identify, What is it, Core rules, 2-Question framework). Don't memorize — understand.
2
Attempt problems blind — Open 02_*_problems.md. Read the problem statement. Try to solve it yourself for 15–20 minutes before looking at the solution.
3
Study the approach table — After seeing the solution, study the approach table (not just the code). Understand each step and why.
4
Re-implement from memory — Close the file. Write the code yourself. Check against the solution. Repeat until fluent.
5
Use the cheatsheet for revision — Section 7 (cheatsheet) in notes files is optimized for quick pre-interview revision. Read it the day before.
